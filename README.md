# APP de Préstamos Pymes

## Descripción
Esta aplicación permite gestionar préstamos de clientes de pequeñas y medianas empresas (PYMES).  
Se pueden registrar clientes, préstamos, letras de cambio y pagos, con la posibilidad de ver un historial completo y controlar el estado de cada préstamo.


## Tecnologías
- **Backend:** Node.js, Express, MySQL2  
- **Frontend:** HTML, CSS, JavaScript  
- **Base de Datos:** MySQL
